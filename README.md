Política de Privacidade - eLivro AI
Última atualização: 02/07/2026
A sua privacidade é importante para nós. Esta Política de Privacidade explica como o aplicativo eLivro AI coleta, usa e protege suas informações.
1. Coleta de Informações
O eLivro AI coleta informações limitadas para fornecer uma experiência de leitura sincronizada e inteligente:
•
Informações de Autenticação: Ao criar uma conta, coletamos seu endereço de e-mail através do serviço Firebase Authentication.
•
Dados de Leitura: Armazenamos metadados dos livros (título, autor) e o seu progresso de leitura (posição do scroll) no Cloud Firestore para que você possa continuar de onde parou em qualquer dispositivo.
•
Interações com IA: As perguntas feitas ao assistente de IA são processadas para gerar respostas. O conteúdo do livro é usado apenas como contexto temporário para a inteligência artificial e não é utilizado para treinar modelos externos.
2. Uso de Dados
Usamos os dados coletados exclusivamente para:
•
Autenticar seu acesso com segurança.
•
Sincronizar sua biblioteca na nuvem.
•
Fornecer insights e respostas inteligentes sobre os livros através do assistente de IA.

4. Segurança
Empregamos medidas de segurança padrão da indústria para proteger suas informações. No entanto, nenhum método de transmissão pela internet ou armazenamento eletrônico é 100% seguro.
5. Exclusão de Dados
Você tem o direito de solicitar a exclusão de sua conta e de todos os dados associados a qualquer momento. Para isso, entre em contato através do e-mail abaixo ou utilize as opções de gerenciamento de conta dentro do aplicativo (quando disponíveis).
6. Alterações nesta Política
Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página ocasionalmente para verificar quaisquer mudanças.
